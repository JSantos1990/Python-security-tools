# 🐍 Python Security Tools

This repository contains a collection of Python scripts focused on cybersecurity automation, log parsing, hashing, metadata analysis, and network discovery.  
All tools are simple, educational, and built to demonstrate practical skills in Python for security purposes.

---

## 📂 Repository Structure

### 📁 `hash-tools`
Scripts for hashing files, comparing digests, and verifying integrity.

### 📁 `metadata-tools`
Tools to extract metadata from images, documents, and other file types.

### 📁 `network-scanners`
Basic network discovery and port scanning tools written in Python.

### 📁 `log-parsers`
Log parsing utilities for SOC workflows, including filtering and IOC extraction.

### 📁 `utilities`
Additional tools and helper scripts for general cybersecurity tasks.

---

## 🔧 Technologies Used
